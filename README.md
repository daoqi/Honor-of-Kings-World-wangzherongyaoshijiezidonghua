## 使用说明

- 基于 OpenCV2 + YOLOv8 图形识别
- 图片基于电脑 16:9 分辨率下截取
- 需要把游戏窗口化 1920x1080 100%缩放，不然可能识图不准确
- **窗口化游戏 ↑ 窗口化游戏 ↑ 窗口化游戏 ↑**
- 窗口可以是任意位置，但必须为窗口化模式
